原码反码补码：
    负数的补码等于 反码+1
    7   原码    00000111
    -7  原码    10000111
    -7  反码    11111000
    -7  补码    11111001
    从-7原码得到-7补码：补码 = (原码求反码)+1
    从-7补码得到-7原码：原码 = (补码-1)求反码
